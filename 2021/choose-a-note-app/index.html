笔记软件的选择

工作这几年，陆陆续续也用了好几款笔记软件了，早几年，这些软件都还是免费的，慢慢的，大家都开始收费了，没办法，都是要生存的。

  • 有道云笔记

    已经没有印象为什么弃了…

  • 为知笔记

    好用好看，存储方便，还有浏览器插件，刚出的时候着实惊艳，不过还没用爽就收费了…

  • 石墨文档

    简单好用的 word 编辑方式，网页版就很够用，当时记录了很多私密信息，完全没有收费迹象，用了好一阵，最终还是收费了…

  • Notion

    有挺多模板的，印象比较深刻的是在页面里再建页面的功能，用的也不少，我用的是学生套餐，也很舒服，可是学生套餐总有过期的时候，继续折腾…

这条路上还有很多诸如 Evernote、 OneNote、 SimpleNote、熊掌记、有道云笔记…还有些是私有云的方式，可能会满足一些对隐私要求高的用户的需求：Leanote 、MarkIdea、为知笔记（又有我- -!），还有些类似nextcloud、坚果云等，方法千奇百怪，最终我还是选择了这套方案：

Typora + Github + PicGo + jsdelivr(加速) + github 客户端（手机端查看） + 脚本自动提交

一条龙服务来实现笔记记录。

当然笔记整理后，接着甩到 Github Pages 就可以输出一篇博客啦~ 岂不美滋滋~

这个方式只是我目前相当满意的方案，不排除后期有更好的选择，也是期待哈，折腾永不止步~

首先说下目前这套方案的缺点：

1、github 单个仓库有容量上限（据说2G），PicGo 上传到了Github仓库，所以有这个隐患，但是到达上限后可以通过再开一个仓库续命，笔记库同理。

2、多台电脑操作未优化，我目前环境是一台电脑，若有两点一线的需求，则需要将脚本添加一个拉去的动作，也是一个小问题，但是存在解决方案。

3、手机端缺少全文搜索，这个只能靠文件分类来弥补了，电脑端是可以的，Typora 支持目录搜索（Ctrl + Shift + F）哦~

接下来就都是优点了：

1、在Typora中做笔记，这就够了…

2、多端同步，电脑，手机全平台

3、Free

4、文件控制在自己手中，虽然放在Github，但是 Github 他会倒吗？他倒的话会没有征兆吗？他会收费吗？他都有免费仓库了…

综上：

开始你的笔记之旅吧~

满足~
