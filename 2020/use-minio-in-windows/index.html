minio 的简单使用

我们农场里正式线使用的对象存储是七牛云，在测试的时候，我们如果再使用普通的上传到服务器，就有可能需要做两套文件系统，于是 minio 就登场啦，他支持 AWS S3 存储协议，在项目里，我们就可以直接使用 aws-java-sdk-s3 的 jar包轻松实现兼容，修改配置就能享用。

在 windows 中部署 minio

windows 下载 .exe 放入自己的文件目录中运行

https://dl.min.io/server/minio/release/windows-amd64/minio.exe

修改密码

在文件目录中会生成一个 .minio.sys 文件夹，进去修改 .minio.sys\config\config.json 文件中的 credentials 的 access_key 和 secret_key 即可修改账户密码。

运行

打开 cmd 进入 minio的文件目录，执行

minio.exe server D:\你的目录

如图得到如图，即成功开启了 minio 服务

enjoy it~
